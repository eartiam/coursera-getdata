Codebook for the Tidy Data Set
==============================

* activity

    Descriptive string for each type of activity

        "LAYING"
        "SITTING"
        "STANDING"
        "WALKING"
        "WALKING_DOWNSTAIRS"
        "WALKING_UPSTAIRS"

* subject

    Subject identifier

        1..30  .Unique identifier for each experiment subject
